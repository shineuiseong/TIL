# node.js 교과서 개정2판 (제로초) Nodebird 예제

Node.js 개정2판 노드버드 프로젝트 입니다.

기능

- 로컬 회원가입 및 로그인
- 카카오 로그인
- 구글 로그인
- 게시글 등록
- 이미지 첨부 및 태그 검색
- 팔로우 기능
- 좋아요 기능 ( 미완 )

#

문제사항

- 좋아요 기능 구현이 완료되지않음

#

필요한 설정

- .env 파일에

  COOKIE_SECRET = cookiesecret

  KAKAO_ID = ??

  GOOGLE_ID = ??

  GOOGLE_SECRET = ??

  추가필요합니다.

  mysql 설정
